You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a JSON response with a random image
  • /random -- returns index.html
  • /multiply?num1=2&num2=3 -- multiplies num1 and num2, returns the result
  • /github?query=users/amehlhase316/repos -- returns a JSON with the full name, ID, and owner login of each repository
  • /concatenate?str1=hello&str2=world -- concatenates str1 and str2, returns the result
  • /power?base=2&exponent=3 -- calculates base raised to the power of exponent, returns the result
